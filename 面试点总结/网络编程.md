# OSI 七层模型

# TCP / IP 四层模型

# 传输层的协议
- **TCP**  : 传输控制协议
- **UDP** : 用户数据报协议

# 应用层的协议
- **DNS**     : 域名系统
- **FTP**      : 文件传输协议
- **Telnet**  : 远程终端协议
- **HTTP**   : 超文本传输协议
- **SMTP**  : 电子邮件协议
- **POP3**  : 邮件读取协议
- **SNMP** : 简单网络管理协议
- **TFTP**   : 简单文件传送协议

# TCP
TCP是一种面向连接的、可靠的、基于字节流的传输层通讯协议
